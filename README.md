# CallSite 🚀

Site vitrine professionnel pour **CallSite** — agence de création de sites vitrines pour artisans, commerçants et indépendants.

## Structure
```
CallSite/
├── index.html       # Page principale
├── css/style.css    # Styles
├── js/main.js       # Scripts
├── netlify.toml     # Config Netlify
└── README.md
```

## Sections
- **Hero** — accroche + mockup navigateur animé
- **Services** — sites vitrines, SEO local, formulaire devis, galerie, maintenance
- **Métiers** — tous secteurs d'activité
- **Portfolio** — 8 faux sites avec prévisualisations CSS (filtres par catégorie)
- **Tarifs** — 3 forfaits (499€ / 799€ / 1199€) + maintenance 29€/mois
- **Process** — 4 étapes
- **Témoignages** — 3 avis clients
- **Contact** — formulaire de devis

## Déploiement
Connecté à Netlify via GitHub. Chaque push sur `main` déclenche un déploiement automatique.
